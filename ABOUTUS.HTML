Your Destination

About "Your Destination" Project


Your destination is a multifacility website that provides the user with suggestions of curated tourist destinations. Website provides the user with VIDEO TOUR and MAP and with the information about HOW TO REACH the destination, but the most prominent feature of "YOUR DESTINATION" is that it can provide the user with a VIRTUAL TOUR of the suggested location and they are also provided with some other location suggestions and activities that they can do and visit in and around the location.
during this project i had to create surveys and produce templates for the website layout . in addition to this i had to collect images of the places, so that they could be shown on the website and finally I produced the actual homepage and content page . This website was built using HTML and CSS.To make the website responsive for Mobile devices , classes and attributes of Bootstrap were used . This website has an animated navigation bar and it's background for the homepage is also animated. Some fonts were taken from google fonts . To animate the website AOS attributes were used .
